# Automatic Visitor Counter using Arduino

## 📌 Project Overview

The Automatic Visitor Counter is an embedded systems project developed using Arduino Uno, an HC-SR04 Ultrasonic Sensor, and a 16×2 I2C LCD Display. The system automatically counts visitors entering and exiting a room by measuring distance and displays the current count on the LCD.

---

## 🎯 Objectives

- Automatically count visitors entering and exiting a room.
- Display visitor count in real time.
- Reduce manual counting errors.
- Provide a simple and low-cost occupancy monitoring system.

---

## 🛠 Hardware Components

- Arduino Uno
- HC-SR04 Ultrasonic Sensor
- 16×2 I2C LCD Display
- Breadboard
- Jumper Wires
- USB Cable

---

## 💻 Software Requirements

- Arduino IDE
- Arduino C/C++

---

## ⚙️ Working Principle

1. The ultrasonic sensor continuously measures the distance.
2. If an object is detected within 20 cm, it is counted as an entry.
3. If an object is detected between 21 cm and 40 cm, it is counted as an exit.
4. The Arduino updates the visitor count.
5. The LCD displays:
   - Entry Count
   - Exit Count
   - Total Visitors
